Tag span e div

A tag span serve para agrupar elementos em linha num documento.
Com essa tag, eu posso estilizar uma parte específica do texto



Enquanto a tag span destaca um trecho de um texto, a tag div agrupa um bloco de tags

A tag mark é usada para destacar partes do texto. É bem feio, mas tem esse recurso

A tag meter representa um valor escalar dentro de um intervalo conhecido ou um valor fracionário.
Por exemplo, Essa barrinha está 65% cheia
